Guitar Chord Chart

This is a bloat-free, ad-free tracking-free website made with one simple puporse: to show guitar chords.

If you would like to contribute to this page visit the GitHub repository.

  • Open and closed string represented with "O" and "X";
  • The numbers at the top represent the fret;
  • A "T" is listed at the top when a string should be played with the thumb;
  • The other numbers represent which finger should be placed where;
  • Here is an example of an A chord:
